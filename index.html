How It Works

1. The system continuously monitors live CCTV footage.

2. AI algorithms analyze video frames in real-time to detect accidents.

3. Upon accident detection, the system extracts the location from the camera's IP address.

4. An alert is triggered with a unique tracking ID and GPS coordinates.

5. Emergency services (ambulance, police) are notified immediately to respond to the accident site.

6. The system logs data for further analysis and improvements.
